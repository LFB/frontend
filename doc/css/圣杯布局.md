## 圣杯布局
圣杯布局是来源于该布局效果类似圣杯而得名，简单来说，就是指三行三列布局。

#### 方法一
- 缺点：位置调整，浮动
```html
<div class="left">left：定宽</div>
<div class="right">right: 定宽</div>
<div class="center">center 自适应</div>
```

```css
.left {
    width: 200px;
    float: left;
    background: red;
}

.right {
    float: right;
    width: 200px;
    background: green;
}

.center {
    margin: 0 200px;
    background: blue;
}
```

#### 方法二：float + margin（经典版）

```html
<div class="parent">
    <div class="center">center 自适应</div>
    <div class="left">left：定宽</div>
    <div class="right">right: 定宽</div>
</div>
```

```css
.parent {
    margin: 0 200px;
}

.left, .center, .right {
    float: left;
}

.center {
    width: 100%;
    background: green;
}

.left, .right {
    width: 200px;
    position: relative;
}

.left {
    margin-left: -100%;
    left: -200px;
    background: red;
}

.right {
    margin-left: -200px;
    right: -200px;
    background: blue;
}
```

#### 方法三：absolute

```html
<div class="left">left：定宽</div>
<div class="center">center 自适应</div>
<div class="right">right: 定宽</div>
```

```css
.left, .center, .right {
    position: absolute;
}

.left {
    width: 200px;
    left: 0;
    background: red;
}

.right {
    right: 0;
    width: 200px;
    background: green;
}

.center {
    left: 200px;
    right: 200px;
    background: blue;
}
```


#### 方法四：table

```html
<div class="parent">
    <div class="left">left：定宽</div>
    <div class="center">center 自适应</div>
    <div class="right">right: 定宽</div>
</div>
```

```css
.parent {
    display: table;
    width: 100%;
}

.left, .center, .right {
    display: table-cell;
}

.left {
    width: 200px;
    background: red;
}

.right {
    width: 200px;
    background: green;
}

.center {
    background: blue;
}
```


#### 方法五：flex

```html
<div class="parent">
    <div class="left">left：定宽</div>
    <div class="center">center 自适应</div>
    <div class="right">right: 定宽</div>
</div>
```

```css
.parent {
    width: 100%;
    display: flex;
}

.left {
    width: 200px;
    background: red;
}

.right {
    width: 200px;
    background: green;
}


.center {
    flex: 1;
    background: blue;
}
```

#### 方法六：grid
```html
<div class="parent">
    <div class="left">left：定宽</div>
    <div class="center">center 自适应</div>
    <div class="right">right: 定宽</div>
</div>
```

```css
.parent {
    display: grid;
    grid-template-columns: 200px auto 200px;
    width: 100%;
}


.left {
    background: red;
}

.right {
    background: green;
}

.center {
    background: blue;
}
```
