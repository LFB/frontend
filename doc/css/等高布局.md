## CSS等高布局
等高布局就是一行被划分为若干列，每一列的高度是相同的值	

#### 方法一：table
```html
<div class="parent">
    <div class="column1">
        column1
    </div>

    <div class="column2">
        column2
    </div>
</div>
```

```css
.parent {
    /*表格的单元格默认是等高的*/
    display: table;
    width: 100%;
}

.column1,
.column2 {
    width: 100px;
    display: table-cell;
}
```

#### 方法二：padding + margin

```html
<div class="parent">
    <div class="column1">
        column1
    </div>

    <div class="column2">
        <p>column2</p>
        <p>column2</p>
        <p>column2</p>
        <p>column2</p>
        <p>column2</p>
        <p>column2</p>
    </div>
</div>
```

```css
.parent {
    overflow: hidden;
}

.column1,
.column2 {
    width: 200px;
    float: left;
    padding-bottom: 9999px;
    margin-bottom: -9999px;
}

.column1 {
    background: red;
}

.column2 {
    background: green;
}
```

#### 方法三：flex
```html
<div class="parent">
    <div class="column1">
        column1
    </div>

    <div class="column2">
        <p>column2</p>
        <p>column2</p>
        <p>column2</p>
        <p>column2</p>
        <p>column2</p>
        <p>column2</p>
        <p>column2</p>
        <p>column2</p>
        <p>column2</p>
        <p>column2</p>
    </div>
</div>
```

```css
.parent {
    display: flex;
    align-items: stretch;
}

.column1,
.column2 {
    width: 200px;
}

.column1 {
    background: red;
}

.column2 {
    background: green;
}
```
